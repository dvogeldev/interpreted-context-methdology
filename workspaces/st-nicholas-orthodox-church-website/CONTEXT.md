# St. Nicholas Orthodox Church Website

A six-stage workspace for developing and launching the St. Nicholas Orthodox Church website using Payload CMS with specific plugins and features.

## Task Routing

| Task Type | Go To | Description |
|-----------|-------|-------------|
| Backend Setup | `stages/01-backend-setup/CONTEXT.md` | Configure Payload CMS backend with collections, fields, access control, and plugins |
| Sample Data Seeding | `stages/02-sample-data/CONTEXT.md` | Create sample records for testing and development |
| Frontend Development | `stages/03-frontend-dev/CONTEXT.md` | Build frontend components, pages, and theme support |
| Real Data Import | `stages/04-real-data-import/CONTEXT.md` | Migrate existing website data to Payload CMS |
| Testing & Launch Prep | `stages/05-testing-launch/CONTEXT.md` | Test functionality, performance, security, and accessibility |
| Deployment/Launch | `stages/06-deployment/CONTEXT.md` | Deploy to production and monitor |
