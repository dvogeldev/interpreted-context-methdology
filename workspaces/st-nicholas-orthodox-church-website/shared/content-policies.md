# Content Policies and Standards

## Appropriate Content Types
- Bulletins: PDF format only, 50MB maximum file size
- Events: Must include start/end dates, event type selection, location details
- Ministries: Requires group classification and chair/contact information
- Sermons: Video library with proper categorization
- General Pages/Posts: Must align with church mission and Orthodox Christian tradition

## Content Requirements
- All content must align with Orthodox Christian tradition and church mission
- Event dates must be valid and in correct chronological order
- Bulletin files must be PDF format and under 50MB size limit
- Ministry listings require valid chair/contact information
- All content must be appropriate for a church audience
- No content that contradicts Orthodox Christian teachings

## Formatting Requirements
- Proper use of heading hierarchy (H1-H6)
- Semantic HTML structure
- Alt text for all meaningful images
- Descriptive link text (avoid "click here")
- Proper color contrast ratios (WCAG 2.2 AA)
- Consistent spacing and layout
