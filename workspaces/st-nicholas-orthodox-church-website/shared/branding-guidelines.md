# St. Nicholas Orthodox Church Branding Guidelines

## Color Palette - "Regal & Traditional"
- Deep Navy: #0A2342
- Burgundy: #722F37
- Gold: #D4AF37
- Warm Cream: #FAF9F6
- White: #FFFFFF
- Charcoal: #2D2D2D
- Slate: #6B7280
- Light Gray: #E5E7EB
- Forest Green: #166534
- Amber: #D97706
- Crimson: #DC2626

## Typography System
- Headings/Display: Playfair Display
  - Weights: 400, 500, 600, 700
  - Sizes: Various sizes for hierarchy
- Body/UI/Text: Open Sans
  - Weights: 300, 400, 500, 600
  - Sizes: Various sizes for readability

## Visual Identity
- Logo Usage: Clear space, minimum size, prohibited alterations
- Iconography: Line style, consistent stroke weight
- Imagery Style: Warm, reverent, high-quality photography
- Button Styles: Primary (Burgundy), Secondary (Gold), Tertiary (Transparent with border)
